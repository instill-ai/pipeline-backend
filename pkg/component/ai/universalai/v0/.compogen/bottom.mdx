

## Example Recipes

Please refer to the part of `type: universal-ai`

```yaml
version: v1beta
variable:
  prompt:
    title: Prompt
    description: "User input message to be sent to the AI model."
    format: string

output:
  ai-output:
    title: AI Output
    value: "${ai.output.data.choices[0]}"

component:
  ai:
    type: universal-ai
    input:
      parameter:
        n: 1
        stream: true
      data:
        messages:
          - content:
            - type: text
              text: ${variable.prompt}
            role: user
    setup:
      model: gpt-4
    task: TASK_CHAT
```
